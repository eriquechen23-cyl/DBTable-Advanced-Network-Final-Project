Project: Final-Network | Author: P77141155 陳燁龍 | Milestone: explanation revised, implementation verified

Plain-Language Method

DBTable is like a library index: choose a few highly discriminative IP bits, use them to jump to a small bucket, then perform exact five-tuple verification only inside that bucket.

Architecture Flow

Rules -> Parser -> Bit Ranking -> Bucket Table

Packet -> Key Extraction -> Candidate Bucket -> Exact Match -> Highest Priority Rule
